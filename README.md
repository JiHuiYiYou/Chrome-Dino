# 栗栗的夏季森林跑酷记  

在蝉鸣织成的绿网里，夏季森林正午的阳光透过层层叠叠的树叶，在地面上投下斑驳的金斑。一只名叫**栗栗**的小松鼠，正蹲在一截覆满青苔的老树根上，尾巴尖不耐烦地轻拍地面——它的目光锁定前方：几块棱角分明的**灰石头**散落在腐叶间，上方枝叶间隐约闪过**蝙蝠**的黑影，而树干上嵌着一个**木质按钮**，纹理被岁月磨得光滑，像块沉默的邀请函。  


## 一、夏日的召唤：木质按钮的秘密  
栗栗的森林生活向来简单：清晨啃松果，午后追蝴蝶，傍晚在树洞里数星。直到三天前，它在老树根后发现这个木质按钮。按钮旁的树皮上刻着歪歪扭扭的字：“按我，开启夏之跑酷——右上角可改名，榜上有名者为王。” 栗栗用爪子按下去，“咔嗒”一声，面前的腐叶突然裂开，露出向下延伸的**木质跑道**，两侧的灌木疯长，蝉鸣陡然尖锐起来。  

跑道尽头立着块**木质排行榜**，钉在老树干上，右上角嵌着个小铜牌，刻着“改名处”——栗栗用爪子蹭了蹭铜牌，竟浮现出自己的名字“栗栗”，旁边还有一行小字：“空格/上键跳跃，下键蹲下加速下落”。它忽然明白：这不是普通的森林，是一场只属于它的夏日挑战。  


## 二、障碍的韵律：蝙蝠、石头与身法  
跑酷的起点是一片开阔的林中空地，栗栗刚跑两步，第一块**灰石头**就从右侧草丛里“骨碌碌”滚出来——它本能地后跳，爪子刚碰到石头顶，就听见头顶传来“吱呀”一声：两只**蝙蝠**从上层的枝桠俯冲而下，翅膀扫过它的鼻尖。栗栗吓得缩成一团，却听见“咚”的一声闷响——下方另一只蝙蝠撞在了石头上。  

“原来如此！”栗栗抖抖毛，忽然懂了规则：  
- **岩石**总在低处滚动，得用**跳跃**越过（空格/上键），踩中就会被弹开；  
- **下层蝙蝠**贴着地面飞，速度快如闪电，只能**跳跃**躲开，绝不能碰；  
- **上层蝙蝠**在头顶枝叶间刷新，像黑色的逗号悬在光斑里，**绝不能跳跃接触**，否则会被缠住尾巴；  
- **下蹲**（下键）能让身体加速下落，像片枯叶般轻盈，刚好能避开上层蝙蝠的俯冲范围。  

它开始练习：面对滚来的石头，后腿发力高高跃起，尾巴在空中绷成直线；看见下层蝙蝠贴地飞来，立刻侧跳；察觉上层蝙蝠的影子晃动，迅速下蹲，让身体贴向地面——腐叶的湿气沾湿肚皮，却换来蝙蝠擦着头顶飞过的风声。  


## 三、夏日的节奏：蝉鸣与跑酷的合拍  
森林的夏季有自己的韵律。栗栗渐渐发现，**蝉鸣的频率就是障碍刷新的信号**：  
- 短促的“知了——”声后，必有石头从右侧滚来；  
- 连续的“吱吱”声里，下层蝙蝠会成双成对掠过；  
- 悠长的“嗡——”声起时，上层蝙蝠便在头顶“刷”地现身。  

它学着用耳朵“读”森林：当蝉鸣如鼓点密集，就收紧肌肉准备连续跳跃；当鸣声渐缓，便趁机调整呼吸。有一次，三块石头连珠炮似的滚来，栗栗在跳跃间隙下蹲，让身体短暂下沉，恰好避开第二块石头扬起的尘土；紧接着上层蝙蝠俯冲，它顺势后仰，尾巴扫过地面激起落叶，蝙蝠扑了个空。  

“身法比力气重要。”栗栗想起老松树洞壁上的刻痕——那是去年冬天它刻下的“稳”字。此刻在夏季的烈日下，这个字忽然有了温度：跳跃是向上的勇气，下蹲是向下的智慧，二者合起来，便是穿越障碍的诗行。  


## 四、排行榜上的名字：从“栗栗”到“夏之影”  
跑酷的尽头是**木质排行榜**，每过一关，栗栗的名字旁就会多一颗松果标记。最初它叫“栗栗”，分数停在“12关”；后来它把右上角的铜牌蹭亮，刻下新名字“夏之影”——这个名字带着森林的热气，像它跑酷时扬起的尘土。  

排行榜的竞争悄然展开：不知何时，树干上多了几个陌生的爪印，分数从“15关”到“20关”不等。栗栗不服气，每天清晨就来练习：它观察蝙蝠的飞行轨迹，发现上层蝙蝠总在树叶最密处刷新；它摸清石头的滚动规律，总结出“右三左二”的间隔。终于有一天，它一口气冲过“28关”，爪印在排行榜上刻下深深的痕迹——“夏之影 28关”，旁边还沾着几片蝙蝠的绒毛。  


## 五、夏日的真谛：与障碍共舞  
当栗栗跑到“30关”时，障碍忽然变了：石头开始斜着滚，蝙蝠的队形从双排变成三角。它有些慌，接连两次被下层蝙蝠擦到尾巴，摔进腐叶堆里。正当它喘着气想放弃时，头顶的蝉鸣忽然停了——一片梧桐叶飘落，正好盖在它面前的一块石头上。栗栗抬头，看见上层蝙蝠正绕着树叶盘旋，像在跳一支笨拙的舞。  

“原来你们也不是敌人。”栗栗忽然笑了。它不再把蝙蝠和石头看作阻碍，而是当作夏季森林的伙伴：石头是大地抛来的绣球，蝙蝠是天空派来的舞伴。它开始享受这种互动：跳跃时看蝙蝠的翅膀在阳光下折射的光，下蹲时听石头滚过落叶的“沙沙”声，连排行榜上的分数都变得不重要了——重要的是每一次爪子与地面的接触，每一次尾巴与空气的摩擦，每一次心跳与蝉鸣的合拍。  


## 尾声：夏日的跑道没有终点  
如今，栗栗依然每天在夏季森林跑酷。它的名字“夏之影”在排行榜上稳居第一，右上角的铜牌被磨得发亮，刻着一行新字：“身法是风的形状”。木质按钮依旧沉默，却在每个清晨准时发出“咔嗒”声，像在说：“来呀，再跑一圈。”  

森林的夏季很长，长到足够栗栗学会所有跳跃和下蹲的技巧；森林的夏季也很短，短到每一次跑酷都像偷来的时光。但栗栗不在乎——它知道，只要爪子还能发力，尾巴还能平衡，蝉鸣还在耳边，这场与蝙蝠、石头、木质跑道的约会，就会永远继续下去。  

毕竟，夏日的森林从不说再见，只说：“下次，换个花样再来。”

# 🦖 2D 无尽跑酷游戏 (Dino Clone)

这是一个使用 **Godot Engine 4.x (C#)** 开发的 2D 无尽跑酷游戏原型。  
本项目旨在复现经典浏览器游戏（如 Chrome Dino）的核心机制，并探索 Godot 中视差背景的高级控制技巧。

## ✨ 核心特性

### 🌌 智能视差背景系统
- **多层级独立控制**：支持任意数量的 `Parallax2D` 层。
- **动态加速**：背景速度随时间推移逐渐加快，模拟游戏难度提升。
- **数学驱动**：基于原始 `Autoscroll` 速度，通过系数自动计算加速度与速度上限，保证各层视觉协调性。
- **硬上限限制**：速度在达到设定阈值后恒定，防止因无限加速导致的物理崩溃。

### 🎮 玩家控制
- **静态角色**：玩家角色在屏幕 X 轴位置固定（仅进行跳跃动作）。
- **物理驱动**：基于 `CharacterBody2D` 的重力和跳跃逻辑。

### 🏗️ 技术架构
- **语言**：C#
- **引擎**：Godot 4.x
- **设计模式**：数据驱动、节点解耦

## 🛠️ 技术实现细节

### Parallax 加速算法
不同于传统的直接设置速度，本项目采用了基于导数的增量控制：

csharp
// 伪代码逻辑
currentSpeed = min(currentSpeed + acceleration * delta, maxSpeed);
layer.Autoscroll = new Vector2(currentSpeed, 0);


- **Acceleration**：基于原始速度的缩放系数，确保不同层加速节奏一致。
- **MaxSpeed**：同样基于原始速度的比例，保证视觉层次感。

### 节点结构

- World (Node2D)
- ├── ParallaxRoot (Node2D)
- │   ├── Parallax2D (远景)
- │   │   ├── Sprite2D
- │   │   └── ParallaxLayerConfig (Node)
- │   ├── Parallax2D2 (中景)
- │   │   ├── Sprite2D
- │   │   └── ParallaxLayerConfig
- │   └── ...
- ├── Player (CharacterBody2D)
- │   ├── Sprite2D
- │   └── AnimationPlayer
- └── WorldManager (Node)
